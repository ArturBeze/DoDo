## Какая логика использовалась для детекции событий.

Логика была выбрана достаточно прямолинейная, с помощью YOLO мы определяли всех людей в кадре (не только в области ROI) и проверяли, находится ли центр человека в границах ROI. Если да то дальше идет проверка с помощью обычных условных операторов. Если с момента когда стол был занят прошло больше "THRESHOLD" секунд, с этого момента стол считается пустым.
Можно было бы помнить состояние стола каждый кадр, но тогда появляется много одинаковой и не нужной информации.
